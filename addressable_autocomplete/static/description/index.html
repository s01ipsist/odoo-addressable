Addressable Autocomplete

Fast, accurate address autocomplete for AU, NZ & the Nordics

Start typing an address on any contact, customer or vendor and pick a real, verified address from the dropdown. Street, city, state, postcode and geo-coordinates are filled in automatically.

A privacy-friendly alternative to Google Places, built on open address data with first-class coverage of Australia, New Zealand and the Nordics.

Free & open source — bring your own key

This connector is free and open source (LGPL-3). It requires an Addressable API key to work, and a free tier is available.

  1. Sign up at addressable.dev and copy your API key (free tier is fine to start).
  2. In Odoo, go to Settings → General Settings → Addressable Autocomplete and paste your key.
  3. Open any contact and start typing in the Street field.

Privacy & data transparency

No data leaves your Odoo instance until you enter an API key. When enabled, the partial address text you type is sent to the Addressable API (over HTTPS) to return matching suggestions. Your API key is stored server-side and is never exposed to the browser.
