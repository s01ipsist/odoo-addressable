Addressable Autocomplete

Fast, accurate address autocomplete for AU, NZ, Canada, the Nordics & Baltics

Start typing an address on any contact, customer or vendor and pick a real, verified address from the dropdown. Street, city, state, postcode and geo-coordinates are filled in automatically.

A privacy-friendly alternative to Google Places, built on open address data.

Features

  • Autocomplete directly on the contact Street field — no new screens to learn.
  • On selection, fills street, city, state, postcode and geo-coordinates automatically.
  • Country-aware: scopes results to the contact's country, or a configurable default.
  • First-class coverage of Australia, New Zealand, Canada, the Nordics and Baltics — and more.
  • Your API key is stored server-side and is never exposed to the browser.
  • Graceful fallback to manual entry when no key is configured.
  • Free, open source (LGPL-3). Tested on Odoo 17, 18 and 19.

Free & open source — bring your own key

This connector is free and open source (LGPL-3). It requires an Addressable API key to work, and a free tier is available.

  1. Sign up at addressable.dev and copy your API key (the free tier is fine to start).
  2. In Odoo, go to Settings → General Settings → Addressable Autocomplete and paste your key.
  3. Open any contact and start typing in the Street field.

Privacy & data transparency

No data leaves your Odoo instance until you enter an API key. When enabled, the partial address text you type is sent to the Addressable API (over HTTPS) to return matching suggestions — nothing else is collected. Your API key is stored server-side and is never exposed to the browser.

Addressable's privacy policy and terms are published at addressable.dev/pages/terms. For questions, contact support@addressable.dev.
